### New Project: Global B2B Sales Pipeline & Engagement Analytics (Power BI + ZoomCharts)

I recently completed a full end‑to‑end analytics project focused on transforming a complex **global B2B sales pipeline dataset** into a clear, interactive, and decision‑ready **Power BI report**.  
The goal was to help sales leaders, account executives, and business stakeholders understand pipeline health, deal progression, and client engagement at a glance.

### What the project is about  
The dataset represents a real-world enterprise environment with thousands of deals, activities, companies, and sales reps across multiple regions and industries.  
My task was to turn this raw data into a **daily decision-making tool** that answers key questions such as:

- How is the pipeline evolving over time?  
- Which industries and regions drive the most value?  
- Where do deals slow down or get stuck?  
- How does engagement (calls, meetings, demos) impact win probability?  
- Which deals are at risk due to inactivity?

### What I built  
Using **Power BI** and **ZoomCharts Drill Down PRO visuals**, I designed a 3‑page interactive report that tells a complete story:

1. **Pipeline Overview**  
   High-level KPIs, time trends, regional/industry breakdowns, and intuitive drilldowns.

2. **Pipeline Flow & Sales Rep Performance**  
   Stage-by-stage funnel movement, bottleneck detection, and rep-level performance insights.

3. **Engagement & Risk (Drill‑Through Page)**  
   Activity timelines, engagement patterns, inactivity risk scoring, and deal-level diagnostics.

### Key features  
- Fully interactive drilldowns (Year → Quarter → Month → Day)  
- Drill‑through to deal-level engagement dashboards  
- High‑risk deal identification using inactivity and value thresholds  
- Activity mix analysis (emails, calls, meetings, demos, workshops)  
- Clean, intuitive layout optimized for both desktop and touch devices  
- Story-driven design that guides users from overview → analysis → action  

###  Why this matters  
Sales teams need more than dashboards—they need **clarity**.  
This report helps them:

- Prioritize deals that need attention  
- Understand where revenue is coming from  
- Improve forecasting accuracy  
- Coach reps based on real performance data  
- Strengthen client engagement strategies  

This project strengthened my skills in **data modeling, DAX, user experience design, and interactive storytelling**—and it’s now part of my analytics portfolio.
