# Global-B2B-Sales-Pipeline-Engagement-Analytics-Power-BI-ZoomCharts-
I recently built an interactive Power BI report that transforms a complex global B2B sales pipeline dataset into a clear, actionable decision‑making tool. The dataset includes thousands of deals, activities, companies, and sales reps across multiple regions and industries.

### What the report delivers
- A full view of **pipeline health**, revenue trends, and regional/industry performance  
- Insights into **deal progression**, bottlenecks, and sales rep effectiveness  
- Analysis of **client engagement** (calls, meetings, demos) and its impact on win probability  
- Identification of **high‑risk deals** based on inactivity and deal value  

### How it’s built
- Power BI with a clean **star schema** (FactDeals, FactActivities, DimCompany, DimSalesRep, DimDate)  
- **ZoomCharts Drill Down PRO visuals** for smooth drilldowns and intuitive exploration  
- A 3‑page story-driven layout:  
  1. Pipeline Overview  
  2. Pipeline Flow & Rep Performance  
  3. Engagement & Risk (Drill‑Through Page)

### Why it matters
The report helps sales teams prioritize deals, improve forecasting, and strengthen client engagement—turning raw data into daily, actionable insights.
